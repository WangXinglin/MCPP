# CodeFlow Prediction Demo

Static GitHub Pages demo for CodeFlow MCPP prediction results.

The page loads sanitized aggregate JSON only:

- `data/manifest.json`
- `data/axes.json`
- `data/queries.json`
- `data/matrices/query_*.json`

The raw execution JSONL files, worker logs, claims, private paths, WebIDE URLs,
API keys, and job metadata are not included.

Current aggregate:

- 462 CodeFlow problems
- 100 budget values
- 100 deadline values
- 4,619,043 complete cells
- 957 missing cells
- 0 partial cells
